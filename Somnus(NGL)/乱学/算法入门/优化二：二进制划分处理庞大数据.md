# 核心思想🤓

- 面对庞大的数量 *n* (如2000)，如何减少计算？
- 答案是将它划分成一个个块，而且这些块的组合要能够表示n以内的所有数据
# 为何可行？🤔


| 取值  |  s(余项,<$2^k$)   |     $2^k$     | ...... |     $2^i$     | ...... |     $2^1$     |     $2^0$     |
| :-: | :-------------: | :-----------: | :----: | :-----------: | :----: | :-----------: | :-----------: |
|  n  |        1        |       1       | 1...1  |       1       | 1...1  |       1       |       1       |
| n-1 |        1        |       1       | 1...1  |       1       | 1...1  |       1       |       0       |
| ... |                 |               |        |               |        |               |               |
|  i  | a<sub>k+1</sub> | a<sub>k</sub> |        | a<sub>i</sub> |        | a<sub>1</sub> | a<sub>0</sub> |
|     |                 |               |        |               |        |               |               |
|  1  |        0        |       0       |   0    |       0       |   0    |       0       |       1       |
**即将数据用二进制表示**  **a<sub>i</sub>$\in$ {0、1}**
**那么一定可以满足**
